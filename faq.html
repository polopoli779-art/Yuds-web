Pertanyaan yang sering ditanyakan player jembod

Bagaimana cara join server? +
Masuk ke menu Server List, pilih server aktif, lalu copy IP & port dan masukkan ke Minecraft kamu.
Apakah server iti gratis? +
Ya, server gratis dimainkan. Namun tersedia fitur tambahan melalui sistem p2w.
Bagaimana melapor pemain? +
Kamu bisa melapor melalui Discord resmi atau hubungi admin yang sedang online.
Apakah bisa main di HP? +
Bisa, selama menggunakan Minecraft Bedrock Edition / launcher hp yang kompatibel.
