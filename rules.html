Server Rules

Harap baca sebelum bermain di server

1. No Cheating

Dilarang menggunakan hack, cheat, mod ilegal atau bug abuse dalam bentuk apapun.

2. Respect All Players

Hormati semua pemain. Toxic, rasis, dan penghinaan keras tidak diperbolehkan.

3. No Griefing

Merusak bangunan atau properti pemain lain tanpa izin akan dikenakan sanksi.

4. No Spamming

Spam chat atau promosi server lain dilarang keras.

5. Follow Staff Instructions

Ikuti arahan staff dan admin untuk menjaga kenyamanan server.
